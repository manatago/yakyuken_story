# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

「ヤクケン王国（Yakyuken Kingdom / じゃんけんキングダム）」のゲームシナリオ・設定資料リポジトリ。コードは含まない純粋な創作ドキュメントプロジェクト。全ドキュメントは日本語で記述する。

**ジャンル**: 異世界転生コメディ。じゃんけん（野球拳）で全てが決まる世界に飛ばされた数学科大学生が、確率論と読み合いで成り上がるストーリー。

## Document Structure

```
characters/      キャラクター設定（01〜05の連番）
ideas/           アイデアメモ・ブレスト資料
world/           世界観設定（法則、社会制度）
scenario/
  flow/          ストーリー進行チャート（章立て・イベント一覧）
  detail/        詳細シナリオテキスト（演出指示・セリフ付き）
```

## Conventions

- **ファイル命名**: `{連番}_{内容}.md`（例: `01_protagonist.md`、`01_world_rules.md`）
- **シナリオテキスト**: `scenario/detail/` 配下は `.txt` 形式。演出指示（`＜キャラ登場＞`）・背景指定（`背景: bg01_xxx.png`）・ナレーション・セリフを含む台本形式
- **設定資料**: `characters/`、`world/`、`ideas/` 配下は `.md` 形式。見出し階層で構造化

## Key World Rules (Consistency Reference)

シナリオや設定を追加・編集する際、以下の根幹設定との整合性を保つこと：

- **脳内チップ**: 全住民は出生時にチップ埋込。身分証＋戦闘インターフェース。主人公Aはチップ未搭載の異世界人→マチルダが旧式チップを注射で後付け
- **カード＝服＝HP**: カード所持数が服の枚数でありHP。5枚未満で逮捕（社会的死）
- **戦闘システム**: 公式戦はHP4勝負（4枚脱がせれば勝利）。ギルド戦はHP1（即決）
- **通貨なし**: 労働（週2日）で衣食住が配給される社会。昇格はギルドランクのみ
- **500年前の真実**: 建国女王＝ヒロインB、技術者（イーロン似）がチップ開発、パパ（PAPA）はその遺産AI
- **主人公の名前**: 設定資料では「A」、詳細シナリオでは「サトシ」。どちらも同一人物
